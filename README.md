# CONTACT • DIP TARAFDER

A modern, interactive **About & Contact page** built with HTML, CSS, and JavaScript.  
This project showcases a stylish profile card with a toggleable panel for detailed contact information.

---

## ✨ Features

- **[Interactive About Button](ca://s?q=Explain_interactive_about_button)**  
  Opens a sleek panel with profile details and contact links.

- **[Animated Background](ca://s?q=Tell_me_about_animated_background)**  
  Blurred, zooming background image with smooth gradient overlay.

- **[Responsive Design](ca://s?q=Explain_responsive_design)**  
  Scales elegantly across desktop and mobile devices.

- **[Profile Panel](ca://s?q=Describe_profile_panel)**  
  Includes profile picture, name, role, and social/contact links.

- **[Modern UI Effects](ca://s?q=Modern_UI_effects)**  
  - Glassmorphism blur effects  
  - Gradient shine animation  
  - Hover transitions and shadows

## 🚀 Usage

1. Clone or download the repository.
2. Open `index.html` in any browser.
3. Tap the **ABOUT** button to reveal the contact panel.

---

## 🖼️ Visuals

- **Profile Card**: Rounded, glowing avatar with hover animation.  
- **Panel Layout**: Central modal with email, phone, and social links.  
- **Background**: Dynamic zoom effect with blur and brightness filter.

---

## 🔧 Customization

- **Profile Image**: Replace the `img` source in both the button and panel.  
- **Colors & Gradients**: Adjust `linear-gradient` values in CSS.  
- **Social Links**: Update `<a>` tags with your own URLs.  
- **Animations**: Modify `@keyframes zoom` and `@keyframes shine` for custom effects.

---

## 📱 Responsive Design

- Optimized for screens down to **480px width**.  
- Profile card and panel scale smoothly on mobile devices.
